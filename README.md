# Integration


### Add this dependency to  ```java build.gradle(Module: app) ```

```java
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```


### And add this to   ```java build.gradle(project:)```

```java
dependencies {
	        implementation 'com.github.MahmoudMostafa125:LibRepo:Tag'
	}
 ```
