# Integration


### Add this dependency to  ``` build.gradle(Module: app) ```

```java
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```


### And add this to   ``` build.gradle(project:)```

```java
dependencies {
	        implementation 'com.github.MahmoudMostafa125:LibRepo:0.1.0'
	}
 ```
