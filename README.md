# Integration


### Add this dependency to  ``` build.gradle(Module: app) ```

```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```


### And add this to   ``` build.gradle(project:)```

```
dependencies {
	        implementation 'com.github.MahmoudMostafa125:LibRepo:Tag'
	}
 ```
